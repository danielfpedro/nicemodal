Nicemodal

Levíssimo plugin de Jquery que carrega uma página interna via ajax em uma janela modal responsiva. Não

Download Nicemodal v1.0

Demo

Clique no botão abaixo para visualizar a janela modal com as opções default. Você pode conferir as opções disponiveis clicando aqui. Após abri-la redimensione o seu navegador ou abra esta página no seu disposito móvel para ver como ela se comporta em telas estreitas.



Como usar

Inclua os arquivos .css e .js do plugin na página.


Inicie o plugin no elemento ou a um grupo deles. O plugin usará como url a ser chamada o que estiver no href para A e o atributo data-url para button. 

$('a#nicemodal, button#nicemodal').nicemodal({
//Veja mais abaixo opções que podem ser incluidas aqui.
});

<a href="pagina.html" id="nicemodal">
    Abre modal
</a>
<button data-url="pagina.html" type="button" id="nicemodal">
    Abre modal
</button>

Se você precisar abrir ou fechar a janela modal via programação basta chamar os métodos públicos do plugin openModal ou closeModal da seguinte forma: 

<!-- Para abrir -->                        
$().nicemodal('openModal', 'contato.html', {//opções aqui});
<!-- Para fechar-->                        
$().nicemodal('closeModal');


Opções

Opção Default Descrição
Width 600px (String) Largura da janela modal, sempre deve ser especificada. Caso a largura do navegador seja menor que a largura especificada ela receberá a medida de 90% com 5% de margem para cada lado.
KeyCodeToClose 27 (Int) Esc Código da tecla que irá fechar a janela modal, caso não queira que nenhuma tecla feche a janela, usar null. Você pode ver ver os códigos das teclas disponíveis neste link.
DefaultCloseButton True (Boolean) Habilita ou sesabilita o botão de fechar padrão do Plugin.
closeOnClickOverlay True (Boolean) Habilita ou desabilita o fechamento da janela modal clicando fora da janela.
closeOnDbClickOverlay True (Boolean) Habilita ou desabilita o fechamento da janela modal dando um duplo click fora da janela.
onOpenModal null (Função) Função executada assim que a janela modal estiver completamente aberta.
onCloseModal null (Função) Função executada assim que a janela modal estiver completamente fechada.
